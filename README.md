# Crack_Detection-using-Tello-Drone

Clone repo 
```
$ git clone https://github.com/KaniskarK/Crack_Detection-using-Tello-Drone.git
$ cd Crack_Detection-using-Tello-Drone
```
Install dependencies
```
$ pip3 install djitellopy
```
